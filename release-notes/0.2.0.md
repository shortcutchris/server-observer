# Server Observer 0.2.0

Server Observer denkt jetzt in Projekten statt nur in einzelnen Ports.

- Überwache mehrere frei wählbare Projektordner.
- Erkenne Git-, Compose-, Dev-Container- und gängige Sprachprojekte automatisch.
- Sieh lokale Prozesse und Docker-Container gemeinsam im passenden Projekt.
- Behalte auch Datenbanken, Worker und Container ohne Webport im Blick.
- Öffne Webdienste und Projektordner direkt oder starte und stoppe Container.
- Nutze die kompakte Kartenansicht oder ziehe das Fenster für eine Projekt-/Detailansicht größer.

Zusätzlich verbessert diese Version die Menüleistenübersicht, Suche, Filter, Einstellungen und die Statusdarstellung für Docker Healthchecks und Ports.
