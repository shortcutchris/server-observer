# Server Observer 0.3.0

Server Observer wird zum lokalen Developer Control Center:

- Projekte und frei definierbare Profile direkt starten, stoppen und neu starten
- automatische Rezepte für Compose, Node.js, Swift, Go und Rust
- `.server-observer.yml` für eigene Befehle, Ports, Services und Healthchecks
- Live-Metriken für CPU, RAM, Netzwerk, Laufzeit und Docker
- Git-Status, HTTP-Healthchecks und intelligente Portkonflikte
- integrierte Logs und lokaler Aktivitätsverlauf
- optionale macOS-Mitteilungen
- neue Menüleistenaktionen, Apple Kurzbefehle und installierbare CLI

Die App bleibt vollständig lokal und überträgt keine Projekt- oder Laufzeitdaten.
